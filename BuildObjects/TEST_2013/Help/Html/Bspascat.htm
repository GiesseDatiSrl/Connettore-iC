Stampa controllo attività

Nome programma: BSPASCAT.DLL
Moduli richiesti: Parcellazione
Report programma: vai ai report

Questo programma permette di eseguire stampe di controllo sulle attività memorizzate nell'archivio, ad esempio per verificare eventuali attività prima di essere fatturate, oppure per accertarsi che tutte le attività siano state fatturate, o ancora per appurate che una singola attività sia stata attribuita a tutti i clienti interessati.

Considera data di esecuzione attività

Se selezionato abilita la possibilità di eseguire stampe da data a data esecuzione attività. Permette quindi di abilitare o disabilitare i campi per inserire la data di inizio/fine esecuzione attività

Codice voce parcellazione

Se indicato permette di stampare solo attività avente il codice parcellazione uguale a quello impostato

Da/A dipendente

I due campi permettono di indicare da che codice a che codice dipendente stampare le attività. Verranno stampate solo le voci con codice dipendente compreso tra quello iniziale e quello finale

Codice cliente

Se impostato diverso da 0 consente di stampare sole le attività assegnate ad un determinato cliente

Da / A data inerimento modifica

Quando si inserisce o si modifica una voce di parcellazione viene memorizzata la data del giorno in cui si è eseguita l'operazione. Tramite queste date è possibile filtrare i movimenti in base alla data effettiva di inserimento/modifica

Nome dipendente

Permette, se impostato, di visualizzare solo i movimenti inseriti o modificati dal un dipendente (nome operatore con cui si è entrati in Business)

Tipo riga

Spuntare i tipi di attività da stampare

Da contratto

Selezionare se stampare solo le voci parcate con il flag 'da contratto', oppure quelle non appartenente a questa tipologia o entrambe. Si fa presente che attualmente il flag non attiva nessun automatimo, ma è solo una informazione da gestire manualmente.

Fatturato

Selezionare se stampare solo le voci fatturate, solo quelle non fatturate o entramble

Emesso avviso di parcella

Selezionare se stampare solo le voci per cui è stato emesso un avviso di parcella, oppure solo quelle per cui non è stato ancora emesso tale documento, oppure entrambi i casi.
